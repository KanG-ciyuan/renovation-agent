# 装修咨询智能客服 Agent

面向装修公司的智能客服与线索跟进系统。用户发送装修咨询后，系统自动完成意图识别、客户画像提取、回复生成、意向评级和飞书同步。

## 能做什么

| 模块 | 功能 |
|------|------|
| 客服回复 | 根据客户意图自动生成首轮回复（支持本地规则 / Coze 智能体两种模式） |
| 线索提取 | 从自然语言中提取面积、预算、风格、房型、地区等关键信息 |
| 意向评级 | 根据信息完整度和关键词自动打分（A/B/C 三级） |
| 飞书推送 | 结构化线索数据通过 webhook 实时推送到飞书群 |

## 快速开始

```bash
# 启动服务
node app.js

# 浏览器打开 http://localhost:3000
```

三种运行模式：

```bash
# 1. 纯本地规则（默认）
node app.js

# 2. 接入 Coze 智能体生成回复
AI_MODE=coze node app.js

# 3. 接入飞书推送
FEISHU_WEBHOOK_URL='https://open.feishu.cn/...' node app.js
```

命令行批量测试：

```bash
node run-demo.js
```

## 技术细节

- **运行时**：Node.js，零第三方依赖
- **意图识别**：关键词匹配 + 正则提取，覆盖量房、报价、风格、工期等常见咨询场景
- **评分算法**：基础分 45，按信息完整度累加（有预算 +20、有面积 +15、有地区 +10、量房预约 +15），上限 100
- **Coze 集成**：通过 `/v3/chat` 接口调用，本地规则作为 fallback
- **飞书集成**：webhook 方式推送文本消息到群聊
