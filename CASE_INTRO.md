# 可直接投递的案例说明

## 案例名称

装修公司智能客服与线索跟进 Agent

## 我做了什么

我围绕装修公司真实咨询场景，独立做了一个可运行的 AI Agent 样板。  
用户发来装修相关问题后，系统会自动完成：

- 识别客户意图，例如价格咨询、量房预约、效果图咨询、工期咨询
- 提取客户关键信息，例如面积、预算、风格、房屋类型、地区
- 自动生成客服首轮回复
- 判断客户意向等级
- 输出销售跟进建议
- 生成可同步到飞书表格或 CRM 的结构化数据

## 我用了什么

- Agent/流程设计思路
- Node.js 落地可运行 demo
- 可扩展到 Coze 多智能体
- 可扩展到飞书 webhook / 飞书表格同步

## 这个案例为什么有价值

它不是单纯聊天机器人，而是贴近商家实际业务的获客与转化工具。  
核心价值在于把自然语言咨询，转成可跟进、可沉淀、可复用的结构化线索。
