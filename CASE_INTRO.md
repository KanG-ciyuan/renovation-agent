# 项目背景

## 要解决的问题

装修公司日常会收到大量客户咨询，包括价格、风格、工期、量房预约等。这些咨询如果靠人工一条条回复、手动整理客户信息，效率低且容易漏单。

这个项目的核心思路是：把自然语言咨询自动转成结构化线索，让销售团队拿到的不再是一段聊天记录，而是可以直接跟进的客户档案。

## 做了什么

围绕装修公司真实咨询场景，实现了从输入到输出的完整自动化流程：

- 识别客户意图（量房预约、价格咨询、效果图咨询、工期咨询、风格咨询）
- 提取关键信息（面积、预算、风格偏好、房屋类型、所在区域）
- 生成客服首轮回复（支持本地规则和 Coze 智能体两种模式）
- 评定客户意向等级（A/B/C 三级，基于信息完整度自动打分）
- 生成结构化数据，通过 webhook 同步到飞书群

## 技术选型

- Node.js 原生实现，零第三方依赖
- 意图识别基于关键词匹配 + 正则提取
- Coze API v3 作为可选智能体后端
- 飞书群机器人 webhook 做消息推送

## 演示方式

- 本地运行：`node app.js`，浏览器访问 `localhost:3000`
- 在线 Demo：https://diagnoses-omnivore-stylized.ngrok-free.dev
- 命令行测试：`node run-demo.js`
